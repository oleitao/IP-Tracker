# IP-Tracker

**Features**
- Retrieve IP or domain geolocation.
- Retrieve your own IP geolocation.
- Retrieve Geolocation for IPs or Domains loaded from file.

**Geolocation Information**
- Status
- Country
- Country Code
- Region
- Region Name
- City
- Zip
- Latitude
- Longitude
- Timezone
- ISP Name
- Organization Name
- AS Number / Name
